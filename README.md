# Hayase-Misa

## 概要

「超時空要塞マクロス」（1982年）に登場するキャラクター、**早瀬未沙 中尉**（後に少佐）のグラフィックを描画するMSX-BASIC プログラムです。

- **バージョン:** 2.0
- **作成日:** 1983年7月25日
- **著作権:** Copyright (C) 1983 by Yoshiyuki Tsuruta

## 動作環境

MSX2 以上（MSX-BASIC）

- スクリーンモード 3（16色グラフィック、640×400ドット相当）
- 配列メモリ: `DIM A(6401)`（スクリーンキャプチャ用バッファ）

## プログラム構成

### メインルーチン

| ラベル | 内容 |
|--------|------|
| `*MAIN.1` | ライン・サークル・ペイントで人物を描画 |
| `*MAIN.2` | 輪郭線の重ね描き・瞳の塗りつぶし |
| `*MAIN.3` | タイトル画面（「Misa Hayase」文字）の描画 |

### 描画サブルーチン

| ラベル | 内容 |
|--------|------|
| `*LINE.1` | 単色折れ線の描画 |
| `*LINE.2` | カラー指定折れ線の描画 |
| `*LINE.3` | C1色での折れ線描画 |
| `*LINE.4` | C2色・ラインスタイル指定の折れ線描画 |
| `*CIRCLE.1` | 円の描画 |
| `*CIRCLE.PAINT.1` | 円の描画＋塗りつぶし |
| `*PAINT.1` | 指定色でのペイント塗りつぶし |

### データセクション

| ラベル | 内容 |
|--------|------|
| `*LINE.D.1` | 人物の輪郭線データ（髪・顔・手・服など） |
| `*LINE.D.2` | 破線・カラー指定の輪郭線データ |
| `*LINE.D.3` | 顔のディテール（鼻・口・首の皺） |
| `*LINE.D.4` | 「Misa Hayase」文字の線データ |
| `*CIRCLE.D.1` | 瞳の円データ |
| `*CIRCLE.D.2` | 瞳のハイライト円データ |
| `*CIRCLE.D.3` | タイトル装飾の円データ |
| `*PAINT.D.1` | 本体の塗りつぶしカラーデータ |
| `*PAINT.D.2` | ズームアップ1段階目の差分ペイントデータ |
| `*PAINT.D.3` | ズームアップ2段階目の差分ペイントデータ |
| `*PAINT.D.4` | ズームアップ3段階目の差分ペイントデータ |
| `*PAINT.D.5` | タイトル画面の背景ペイントデータ |

## 描画の流れ

```
起動
 └─ MAIN.3（タイトル画面: 「Misa Hayase」文字 + 赤背景）
     └─ END（現在の終了位置）

※以下は現バージョンでは未実行（END以降のコード）
    └─ MAIN.1（全身描画）
    └─ PAINT.D.2（顔アップ 差分ペイント）
    └─ MAIN.2（輪郭重ね描き）
       ︙  ズームアップ × 3段階
```

## 座標系

`WINDOW` コマンドで論理座標を設定し、`VIEW` でビューポートを指定しています。

- タイトル画面: `WINDOW(850,950)-(3250,1550)`
- 全身描画: `WINDOW(200,900)-(3400,2900)`
- ズームアップ段階に応じてウィンドウを変更
